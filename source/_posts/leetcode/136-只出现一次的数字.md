# 136. 只出现一次的数字

> **题目链接：**[136. 只出现一次的数字](https://leetcode.cn/problems/single-number/)

## 题目描述

给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

## 核心思考：异或（XOR）运算的神奇消除

要做到不仅 O(N) 时间而且严格禁止申请任何额外存储空间的 O(1) 限制，哈希表（字典和集合）之类的做法全部被判出局。此时必须请出位运算中的大杀器：**异或 (XOR)**。

异或运算有三条针对这道题“量身定做”的黄金定律：
1. 任何数和 0 异或，结果必定是它本身：`a ^ 0 = a`
2. 任何数和它自身异或，结果必定是 0：`a ^ a = 0`
3. 异或运算严格满足交换律和结合律：`a ^ b ^ a = (a ^ a) ^ b = 0 ^ b = b`

综上，只要将数组里所有数字全部挨个进行一遍异或运算。因为除了某个落单分子，其他所有人都是“成双成对”出现的，一对对的相同数字会全部因为异或而互相同归于尽变成 `0`。最后剩下的，就是那个孤零零一次露脸的目标数字。
在 Python 里，可以极致优雅地借用内置函数 `functools.reduce` 结合 `operator.xor` 一行直接连卷消底。

## 解题思路 (Python)

```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        return reduce(xor, nums)

```

## 复杂度分析

- **时间复杂度**：O(N)，N 为数组的长度。完全遍历扫描进行聚合运算。
- **空间复杂度**：O(1)，纯位运算消除，绝对零额外空间申请。
