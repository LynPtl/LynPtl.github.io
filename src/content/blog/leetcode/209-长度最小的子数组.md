# 209. 长度最小的子数组

> <strong>题目链接：</strong>[209. 长度最小的子数组](https://leetcode.cn/problems/minimum-size-subarray-sum/)

## 题目描述

给定一个含有 <code>n</code> 个正整数的数组和一个正整数 <code>target</code>。找出该数组中满足其和 <code>≥ target</code> 的长度最小的 <strong>连续子数组</strong> <code>[numsl, numsl+1, ..., numsr-1, numsr]</code>，并返回其长度。如果不存在符合条件的子数组，返回 <code>0</code>。

## 核心思考：滑动窗口的高效收缩

这道题目是经典的<strong>滑动窗口 (Sliding Window)</strong> 应用。我们的目标是找到一个满足条件的最小窗口。

与“固定窗口”不同，这里的窗口大小是动态变化的。核心逻辑在于：
1.  <strong>右指针 (<code>right</code>)</strong> 不断向右移动以扩大窗口，直到满足（或尽可能接近）条件。
2.  <strong>左指针 (<code>left</code>)</strong> 在满足条件的前提下，不断向右移动以缩小窗口，从而寻找“最小长度”。

### 优化点
在代码实现中，我们可以通过判断 <code>s - nums[left] >= target</code> 来决定是否可以安全地弹出左侧元素，这样可以保证在更新 <code>ans</code> 之前，窗口已经收缩到了满足条件的最小状态。

## 解题思路 (Python)

``<code>python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        n = len(nums)
        ans = n + 1  # 初始化为一个不可能达到的最大值
        s = 0        # 当前窗口内的元素之和
        left = 0     # 窗口左指针
        
        for right, x in enumerate(nums):
            s += x   # 右指针向右移动，累加当前数值
            
            # 如果减去窗口最左侧的数后，和依然大于等于 target
            # 说明我们可以进一步收缩窗口以寻求更小的长度
            while s - nums[left] >= target:
                s -= nums[left]
                left += 1
            
            # 只要当前窗口和满足条件，就尝试更新最小长度
            if s >= target:
                ans = min(ans, right - left + 1)
                
        return ans if ans <= n else 0
</code>`<code>

## 复杂度分析

- <strong>时间复杂度</strong>：$O(n)$，其中 $n$ 是数组的长度。虽然嵌套了 </code>while<code> 循环，但每个元素最多被访问两次（进入窗口一次，离开窗口一次）。
- <strong>空间复杂度</strong>：$O(1)$，仅使用了几个辅助变量。

## C++ 实现

在 C++ 中，我们可以使用类似的逻辑。注意处理和可能超过 </code>int<code> 范围的情况（虽然本题 </code>target<code> 和元素均为正整数且范围适中，但使用 </code>long long<code> 或 </code>int<code> 均可，取决于题目约束）。

</code>`<code>cpp
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int n = nums.size();
        int left = 0;
        int sum = 0;
        int ans = n + 1;
        
        for (int right = 0; right < n; right++) {
            sum += nums[right];
            // 标准滑动窗口收缩逻辑
            while (sum >= target) {
                ans = min(ans, right - left + 1);
                sum -= nums[left++];
            }
        }
        
        return ans > n ? 0 : ans;
    }
};
</code>``
