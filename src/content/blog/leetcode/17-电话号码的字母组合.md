# 17. 电话号码的字母组合

> <strong>题目链接：</strong>[17. 电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

## 题目描述

给定一个仅包含数字 <code>2-9</code> 的字符串，返回所有它能表示的字母组合。答案可以按任意顺序返回。

给出数字到字母的映射如下（与电话按键相同）。注意 <code>1</code> 不对应任何字母。

## 核心思考：回溯法（DFS 枚举）

这道题需要生成所有可能的字母组合，是一道典型的排列组合枚举问题，适合用回溯法（Backtracking）来解决。

每个数字对应若干个字母，我们需要从左到右依次处理输入字符串 <code>digits</code> 中的每一个数字，并在该数字对应的所有字母中逐一尝试，然后递归处理下一个数字位。

具体流程如下：
1. <strong>建立映射表</strong>：用一个数组 <code>values</code> 将数字 <code>0-9</code> 分别映射到其对应的字母字符串上。其中 <code>0</code> 和 <code>1</code> 不对应任何字母，为空字符串。
2. <strong>递归枚举</strong>：定义递归函数 <code>dfs(i)</code>，表示当前正在处理第 <code>i</code> 个数字。对于 <code>digits[i]</code> 对应的每一个字母，将其填入路径数组 <code>path[i]</code> 中，然后递归调用 <code>dfs(i + 1)</code> 处理下一位。
3. <strong>收集结果</strong>：当 <code>i == n</code>（即所有数字都处理完毕）时，将当前路径 <code>path</code> 拼接成字符串，加入结果列表 <code>ans</code> 中，然后返回上一层继续尝试其他字母。

值得注意的是，这里使用了一个固定长度的 <code>path</code> 数组而不是动态拼接字符串，避免了每次递归都创建新字符串对象的开销。

## 解题思路 (Python)

``<code>python
values = ["", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"]
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        n = len(digits)
        if n == 0:
            return []
        
        ans = []
        path =  [""]*n

        def dfs(i):
            if i == n:
                ans.append("".join(path))
                return
            else:
                for char in values[int(digits[i])]:
                    path[i] = char
                    dfs(i+1)
        
        dfs(0)
        return ans
</code>`<code>

## 复杂度分析

- <strong>时间复杂度</strong>：$O(3^m \times 4^n)$，其中 $m$ 是输入中对应 3 个字母的数字个数（如 </code>2<code>、</code>3<code>、</code>4<code> 等），$n$ 是对应 4 个字母的数字个数（如 </code>7<code>、</code>9<code>）。每个数字对应的字母都需要逐一枚举，最终生成的组合总数即为各位字母数量的乘积。
- <strong>空间复杂度</strong>：$O(m + n)$，即输入字符串的长度。递归调用栈的深度等于 </code>digits<code> 的长度，</code>path` 数组的长度也与之相同。不计入存储结果的空间。
