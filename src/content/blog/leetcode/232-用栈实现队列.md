# 232. 用栈实现队列

> <strong>题目链接：</strong>[232. 用栈实现队列](https://leetcode.cn/problems/implement-queue-using-stacks/)

## 题目描述

请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（<code>push</code>、<code>pop</code>、<code>peek</code>、<code>empty</code>）。

## 核心思考：双栈倒腾模拟队列

栈是后进先出（LIFO），队列是先进先出（FIFO）。要用栈模拟队列，关键在于如何将元素的顺序翻转一次，使得最早进入的元素能最先被取出。

做法是维护两个栈：
- <strong><code>s1</code>（输入栈）</strong>：所有新元素都 <code>push</code> 到这里。
- <strong><code>s2</code>（输出栈）</strong>：所有 <code>pop</code> 和 <code>peek</code> 操作都从这里取。

当 <code>s2</code> 为空、需要取元素时，把 <code>s1</code> 中的所有元素逐个弹出并压入 <code>s2</code>。经过这次倒腾，<code>s1</code> 中最早进入的元素（栈底）就跑到了 <code>s2</code> 的栈顶，顺序刚好翻转，符合队列的先进先出语义。

各操作的具体逻辑：
- <strong><code>push(x)</code></strong>：直接压入 <code>s1</code>。
- <strong><code>peek()</code></strong>：如果 <code>s2</code> 为空，先将 <code>s1</code> 全部倒入 <code>s2</code>，然后返回 <code>s2</code> 栈顶元素。
- <strong><code>pop()</code></strong>：先调用 <code>peek()</code> 确保 <code>s2</code> 非空，然后弹出 <code>s2</code> 栈顶。
- <strong><code>empty()</code></strong>：当且仅当 <code>s1</code> 和 <code>s2</code> 都为空时，队列为空。

## 解题思路 (Python)

``<code>python
class MyQueue:

    def __init__(self):
        self.s1 = []
        self.s2 = []

    def push(self, x: int) -> None:
        self.s1.append(x)

    def pop(self) -> int:
        self.peek()
        return self.s2.pop()

    def peek(self) -> int:
        if not self.s2:
            while self.s1:
                self.s2.append(self.s1.pop())
        return self.s2[-1]

    def empty(self) -> bool:
        return not self.s1 and not self.s2


# Your MyQueue object will be instantiated and called as such:
# obj = MyQueue()
# obj.push(x)
# param_2 = obj.pop()
# param_3 = obj.peek()
# param_4 = obj.empty()
</code>`<code>

## 复杂度分析

- <strong>时间复杂度</strong>：</code>push<code> 和 </code>empty<code> 为 $O(1)$。</code>pop<code> 和 </code>peek<code> 的均摊时间复杂度为 $O(1)$。虽然单次倒腾操作可能需要 $O(N)$ 时间，但每个元素在整个生命周期中最多只会被从 </code>s1<code> 移动到 </code>s2` 一次，因此分摊到每次操作上仍然是常数时间。
- <strong>空间复杂度</strong>：$O(N)$，其中 $N$ 是队列中的元素数量。两个栈合计存储了所有元素。
