q1的python-stdin实现

用startswith来匹配

```python
#!/usr/bin/env python3

import sys

count = 0

for line in sys.stdin:
    field = line.strip().split("|")

    if len(field) >=2 and field[-2].startswith("3711/"):
        count += 1

print(count)
```
