# 121. 买卖股票的最佳时机

> **题目链接：**[121. 买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

## 题目描述

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。
你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。
返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

## 核心思考：一次遍历与贪心思想

股票只能买卖一次，为了获得最大利润，我们不需要试图去枚举所有的买卖组合。我们只需要关心一件事：
**如果在第 `i` 天卖出股票，怎样才能赚的最多？**
答案是显而易见的：我们在第 `i` 天之前价格最低的那一天买入。

因此，我们只需要一次扫描（One Pass）整个数组：
1. 设立一个变量 `min_`，用于时刻记录“到今天为止”出现的**历史最低价格**。
2. 每天都先假装如果在今天卖出（即 `当前价格 - min_`），算一算是赚还是亏，不断尝试更新历史最大利润 `ans`。
3. 随后，更新 `min_` 为今天和昨天历史最低之间的较小者，继续往后走。

## 解题思路 (Python)

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        ans = 0
        min_ = prices[0]
        for x in prices:
            ans = max(ans, x - min_)
            min_ = min(min_, x)
        return ans

```

## 复杂度分析

- **时间复杂度**：O(N)，其中 N 为数组长度。只需要一遍循环线性遍历即可得出结果。
- **空间复杂度**：O(1)，单纯使用了两个变量标定价格，内存开销是常数量级。
