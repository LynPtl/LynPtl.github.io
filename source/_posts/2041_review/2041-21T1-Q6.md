需要注意空格。注意strip

```shell
#!/usr/bin/env python3

import sys

for line in sys.stdin:
    field = line.strip().split('|')
    # 需要提前找好空格的长度
    raw_name_len = len(field[2].rstrip())
    space_len = len(field[2]) - raw_name_len

    name = field[2].strip().split(', ')
    name = name[::-1]
    newname = " ".join(name)
    field[2] = newname + " " * space_len
    print("|".join(field))
```
